MyoWare 2.0 → ESP32 · Wiring

Drop-in replacement for the AD8232 setup. Three wires per sensor. No leads-off detection, no cable jack, no LO+/LO− on the ESP32. Same packet format, same Pi-side code.

Recommended for v1: 1 MyoWare only (CH1 / GPIO 34). For open/close control via predict_open_close.py, a single sensor on the forearm flexor is enough. Flex = close, relax = open. Wire only the top sensor in the diagram below — the CH2 connections (GPIO 35, dashed below) are optional and only needed if you later add a 2nd MyoWare for distinct open/close commands.

1 · The change at a glance

Was — AD8232 (per channel)

3.3V→ ESP32 3V3
GND→ ESP32 GND
OUTPUT→ ESP32 GPIO 34 / 35
LO+→ ESP32 GPIO 25 / 26
LO−→ ESP32 GPIO 27 (CH1 only)
3.5 mm jack → electrode cable → snap leads

Now — MyoWare 2.0 + Cable Shield (per channel)

VIN→ ESP32 3V3
GND→ ESP32 GND
SIG→ ESP32 GPIO 34 / 35
no LO+, no LO−
Cable Shield → 3 snap leads → electrodes
Net effect: GPIO 25, 26, 27 are now unused on the ESP32 (they were leads-off detection pins). Your FSR wiring on GPIO 33 / 36 / 39 stays exactly the same. The 54-byte UDP packet format is unchanged.

2 · Wiring map · MyoWare 2.0 × 2 → ESP32

Two MyoWare 2.0 modules, each with a Cable Shield carrying snap leads to disposable electrodes on the forearm. Three wires per sensor go to the ESP32: V+, GND, SIG. Power + ground are shared rails (one each).

3 · Copy-paste connection table

Wire colourFromToNotes
red MyoWare #1 · V+ ESP32 · 3V3 shared rail to MyoWare #2 V+
black MyoWare #1 · GND ESP32 · GND shared rail to MyoWare #2 GND
orange MyoWare #1 · SIG ESP32 · GPIO 34 flexor (close-hand) channel
blue MyoWare #2 · SIG ESP32 · GPIO 35 extensor (open-hand) channel
— MyoWare · RAW (both) — (not wired) RAW is the un-rectified EMG; only needed if you switch back to the full CNN pipeline. For open/close demo use SIG only.
— Cable Shield snap leads (each MyoWare) — (skin) MID → muscle belly, END → 2 cm distal along the fibre, REF → bony spot
⚠ Don't share REF between the two MyoWare sensors unless you also add the MyoWare 2.0 Link Shield. Each sensor expects its own reference electrode. Two REF pads on the same bony spot (e.g. one on each side of the elbow) is fine and what most people do.
